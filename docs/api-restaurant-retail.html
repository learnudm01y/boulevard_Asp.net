Boulevard API — Restaurant, Retail, Laundry & Photography

Mobile Developer Reference  ·  All endpoints use GET  ·  All responses are JSON

Base URL: https://your-domain.com   |   All responses return: { "result": [...], "code": 200, "message": "success", "isSuccess": true }
ℹ️ Every response has the same wrapper: result = the data, isSuccess = true/false, code = 200 (success) or 406 (error). Always check isSuccess before using result.

⚠️ Note: category-products and brand-products return result as a single object (category/brand info + nested products: [...] array), not as a plain array.

Get products that belong to a specific category. Response: result is an object containing category info + products: [...] array.

🧺 LAUNDRY

Laundry Endpoints

ℹ️ Laundry is a Service-type section. All endpoints use featureCategoryId = 21. Unlike Restaurant/Retail (product-based), service sections share the /api/v1/Service/ prefix with all other service categories.

⚠️ GetServices is a POST endpoint — test it via Postman or curl with a JSON body.
POST /api/v1/Service/SearchingServiceType

Keyword search for service types (providers) in the Laundry section. Note: keyword must be non-empty to get results — use GetServices (below) for full listing.

ParameterTypeRequiredDescription
featureCategoryIdintRequiredFixed: 21 (Laundry)
keywordstringRequiredSearch keyword — must be non-empty for results
sizeintOptionalItems to return — default: 10
countintOptionalOffset — default: 0
memberIdintOptionalPass to mark favourites
langstringOptionalen or ar
POST /api/v1/Service/GetServices

Get paginated services list with optional search. Returns service packages for the Laundry section.

Content-Type: application/json

Body FieldTypeRequiredDescription
FeatureCategoryIdintRequiredFixed: 21 (Laundry)
SearchKeywordstringOptionalSearch by service or provider name
sizeintOptionalPage size — default: 10
countintOptionalOffset — default: 0

Example body: {"FeatureCategoryId":21,"SearchKeyword":"","size":10,"count":0}

GET /api/v1/Service/GetServiceDetailsById

Get full details for a specific laundry service, including service types, images, pricing, and amenities.

ParameterTypeRequiredDescription
serviceIdintRequiredService ID (e.g. 50094 for default Laundry Service)
memberIdintOptionalPass to check membership discounts
langstringOptionalen or ar
GET /api/v1/Service/GetRelatedServices

Get other laundry services related to the currently viewed service (for the service detail screen).

ParameterTypeRequiredDescription
featureIdintRequiredFixed: 21 (Laundry FeatureCategoryId)
serviceIdintRequiredThe currently viewed service ID
langstringOptionalen or ar
GET /api/v1/Service/GetFilterResponse

Get available filter options (amenities, types, price ranges) for the Laundry listing screen.

ParameterTypeRequiredDescription
featureCategoryIdintRequiredFixed: 21 (Laundry)
langstringOptionalen or ar
GET /api/v1/Service/GetServiceAmenities

Get all amenity tags available for laundry services (for filtering chips on the listing screen).

ParameterTypeRequiredDescription
featureCategoryIdintRequiredFixed: 21 (Laundry)
langstringOptionalen or ar

POST /api/v1/Order/InsertOrderRequestService

Place a new laundry service order request. Shared by all service sections (Laundry, Photography, Salon, Medical…).

Content-Type: application/json — Requires auth token

Body FieldTypeRequiredDescription
